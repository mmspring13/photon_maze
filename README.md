# Photon Maze

Photon Maze is a minimalist, high-contrast grid puzzle. Slide your energy node through complex fiber-optic networks to restore connection.

Once your node starts sliding, it only stops when it hits a solid wall, requiring logical planning to reach the target receptor.

## Key Features
- **18 Polished Levels**: Split across progressive sectors.
- **Star Rating**: Complete grids under the optimal move limit.
- **Dynamic Themes**: Personalize the interface with vibrant high-contrast accents.
- **Undo & Restart**: Fail safely and experiment with routes.
- **Multilingual Support**: Available in English and Russian.
